# 🐾 PetApp3 – Taro Edition  
**Portable Demo Version (Dog)**  

---

### 🇬🇧 English Description

The **Taro Edition** is the official demo version of PetApp3.  
It includes pre‑generated images and videos of **Taro**, a friendly mixed‑breed dog, so users can experience PetApp3 immediately without preparing their own materials.

---

#### 📘 What’s Included
- Pre‑generated images for n1–p11  
- Pre‑generated affectionate video for p12  
- Pre‑configured pet profile (species, appearance, affection word)  
- Ready‑to‑use PlayWindow setup  

---

#### 📘 How to Use
1. Download the demo ZIP from Releases.  
2. Extract it to any folder.  
3. Run **`run.bat`**.  
4. Taro will appear and react in real time.  

---

#### 📘 System Requirements
- Windows 10 / Windows 11  
- No GPU required  
- Python included (portable)

### 🇯🇵 日本語説明

**PetApp3 – Taro Edition（ポータブル版）** は、  
AIペットアプリ「PetApp3」の公式デモ版です。  
雑種犬「Taro」の画像・動画があらかじめ生成されており、  
ユーザーは素材を準備せずにすぐに体験できます。

---

#### 📘 内容
- 状態 n1〜p11 の画像を同梱  
- p12（愛情反応）の動画を同梱  
- ペットプロフィール（種別・外見・魔法のことば）を設定済み  
- PlayWindow によるリアルタイム反応が可能  

---

#### 📘 使い方
1. ZIPファイルをダウンロードします。  
2. 任意のフォルダに展開します。  
3. **`run.bat`** を実行します。  
4. Taro がリアルタイムで反応します。  

---

#### 📘 動作環境
- Windows 10 / Windows 11  
- GPU不要  
- Python同梱（ポータブル動作）
